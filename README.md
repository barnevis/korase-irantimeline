## irantimeline with Korase
A compact project for irantimeline.

### Steps to Run:
1. Save the script as `install.sh`.
2. Give execution permissions:  
   ```bash
   chmod +x install.sh
   ```
3. Run the script:
   ```bash
   ./install.sh
   ```
